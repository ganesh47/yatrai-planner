# Security Policy

## Supported Versions
We support security fixes for:
- The default branch (`main`).
- The most recent release tag.

## Reporting a Vulnerability
Please do not open public issues for security findings.

Use GitHub Security Advisories:
https://github.com/ganesh47/yatrai-planner/security/advisories/new

Include:
- A clear description of the issue and impact.
- Reproduction steps or a proof of concept.
- Affected versions or commit hashes.
- Any suggested remediations.

We aim to acknowledge reports within 72 hours and provide a remediation plan as soon as possible.
